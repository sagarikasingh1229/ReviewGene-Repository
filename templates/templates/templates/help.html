Frequently Asked Questions

What file formats are supported?

We support Excel files (.xlsx, .xls) and CSV files. Make sure your file has the required columns.

What are the required columns?

Your file must have these 7 columns: sku_id, Name, brand, product_discount_category, Classifier 1, classifier 2, classifier 3.

How many reviews are generated per product?

We generate 17-22 reviews per product, ensuring comprehensive coverage and variety.

What languages are supported?

We generate reviews in English (60%), Hinglish (30%), and Hindi (10%) for authentic Indian context.

How long does generation take?

Generation time depends on the number of products. For 10 products, it typically takes 2-3 minutes.

Are the usernames unique?

Yes! We generate diverse usernames including Indian names, nicknames, alphanumeric combinations, and more.

Troubleshooting

File Upload Error

Problem: File upload fails or shows error

Solution:

  • Check file format (only .xlsx, .xls, .csv supported)
  • Ensure file size is under 10MB
  • Verify all required columns are present
  • Check for special characters in column names

Generation Takes Too Long

Problem: Review generation is taking a very long time

Solution:

  • Check your internet connection
  • Reduce the number of products (try with 5-10 first)
  • Ensure your OpenAI API key is valid
  • Check if you have sufficient API credits

Wrong File Format

Problem: System says file format is incorrect

Solution:

  • Download our sample file to see the correct format
  • Ensure column names match exactly (case-sensitive)
  • Remove any extra spaces in column names
  • Check that data starts from row 2 (row 1 should be headers)
